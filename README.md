数据区含如下内容：
1：类型：App名字
格式为：GBK格式包含结束符0
2：标题
格式为：GBK格式包含结束符0
3：时间：
格式为：20210422T175301的ASCALL码包含结束符0；表示2021年4月22日17点53分01秒
4：信息：
格式为：UTF-8格式包含结束符0
